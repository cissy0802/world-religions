## 定位
中立、比较地理解人类主要信仰传统。每期一个传统，讲清它的历史、今日分布、信仰体系、信徒实践。分工：**philosophy=思想体系**（法拉比、印度六派、老子孔子作为思想家）；**buddhism=佛学深读**（佛学不在本站）；本站讲**活的信仰传统**——历史/分布/信仰/信徒实践，是「世界宗教 101」不是哲学。

## 中立性硬约束（最高优先级）
- **不站队、不传教、不排座次**：呈现每个传统的内部逻辑与最强自我表述(steelman)，让读者自己判断。
- **不评价在任宗教人物、不预测时事、不追热点**；只讲结构性、经得起时间的内容。
- **争议处呈现多方**（教派冲突、以巴、涉华宗教、政教关系等）一律**学理化、多视角**，不褒不贬。
- **数字与史实必须 WebSearch 核实**；人口数给区间（宗教 vs 民族身份定义不同）；不确定的年代/数据/轶事 → 不写。人名仅限广为人知者。
- 每页末尾放一句中立声明（见基准页）。

## 每期结构（四维卡）
1. header：kicker(世界宗教 · World Religions) + h1(中英传统名) + meta(Day N · 所属传统群 · 人口概数)
2. `.lede`：一句话定位——这个传统为何值得理解、一个反直觉钩子。若该传统有核心经文/宣告，可放一句 `.shema` 式引文 + 出处。
3. **四维卡（每维一个 `.card`，`.tag` 标注维度）**，顺序固定：
   - **【历史与起源】** 创始/演化、关键转折、传承脉络；配一张 SVG 时间线（历史长的传统尤其需要）。
   - **【今日分布】** 真实数字（WebSearch 核实、给区间）、地理集中处、增长/衰退趋势；"民族-宗教"还是"普世传教型"要点破。
   - **【核心信仰体系】** 神观、世界观、救赎/解脱观、核心经典；与邻近传统的关键分野。
   - **【信徒的重要原则与实践】** 诫命/戒律、礼拜/祈祷、节期、生命周期礼、伦理原则；内部支派/教派的差异。
4. `.misread` **【常见误解】** 3–4 条（`<b>误解…</b>` → `<span>实为…</span>`），破除大众对该传统的典型误读。
5. `.bridge` **【跨传统对读 · 当代处境】** 与其他传统的关系（同源/影响/对比）+ 当代真实处境与张力。真联系才写，牵强不写。
6. `.thinking` **深入思考** 3 个 `<details>` 开放问题，答案 100–200 字思路。
7. 中立声明（`.neutral`）。

## 图（硬规则）
- **一律画 SVG，绝不用等宽 ASCII 图。** 贴合羊皮纸学者调、与基准页协调。
- 常用图：历史时间线、地理分布、教派谱系树、信仰结构图（如三位一体、五功、种姓四行期）。
- **箭头一律用 `marker-end` + `orient="auto"`，别手画三角。** marker 用 `markerUnits="userSpaceOnUse"`。
- **连杆（线/路径）绝不套 `filter` 辉光或 `objectBoundingBox` 渐变**（轴对齐直线包围盒为 0 会导致整条线不渲染）。要辉光就单独垫一条无 marker 的粗线。
- 布局不挤不乱、文字不重叠不出界、箭头/连线渲染后都真实可见、对比够。
- **画完真渲染检查**（headless Chromium 截图看一眼）：箭头贴线、无浮空三角、无消失连杆。

## 文风（地道中文，硬规则）
- 中立、比较、克制；把术语当读者不懂，随写随用大白话解释（梵/巴利/希伯来/阿拉伯术语首次出现给中文释义）。
- 地道中文，无翻译腔（❌「sfumato blurred 了 contour」）。没有公认译名的术语保留原文并释义。
- 无元叙述：不写"这一期我们讲…""我给你介绍…"，正文只留内容。
- 不假设 BigCat 的年龄/性别/信仰立场。
